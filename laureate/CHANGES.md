# Что добавлено в этой версии

## Новые возможности

1. **Импорт PhD из Excel** (`/new/phd/import`).
   Excel-шаблон: «Шаблон.xlsx» (52 колонки). Каждая строка = один диплом.
   Поддерживается batch-импорт (4+ слушателей).

   **NEW**: Поля «Научные консультанты» и «Рецензенты» теперь корректно
   обрабатывают **до 4 человек на одно поле** — текст автоматически
   разбивается по 2 ФИО в строку (по 2 строки выделено в шаблоне).
   Шрифт автоматически уменьшается, если ФИО длинные.

2. **Сертификат ФДО** (педагогическая переподготовка).
   - Форма: `/new/fdo`
   - Импорт Excel: `/new/fdo/import` (шаблон «ФДО_шаблон.xlsx»)

3. **Сертификат Минор** (к диплому бакалавра).
   - Форма: `/new/minor`
   - Импорт Excel: `/new/minor/import`

## Как теперь обрабатываются 4 ФИО в одном поле

Excel-ячейка с переносами строк:
```
Иванов И.И. - доктор биологических наук;
Петров П.П. - кандидат биологических наук;
Сидоров С.С. - доктор философии;
Козлов К.К. - доктор PhD.
```

Превращается в PDF-блоке в:
```
[Строка 1] Иванов И.И. - доктор биологических наук; Петров П.П. - кандидат биологических наук
[Строка 2] Сидоров С.С. - доктор философии; Козлов К.К. - доктор PhD
```

Шрифт автоматически уменьшается (с 9pt вплоть до 6pt) чтобы текст влез в ширину блока.

Поддерживаемые разделители (в Excel-ячейке): `\n` (Alt+Enter), `;\n`, `;`.
